# FestivalSale


Festival Sale is a fluid web component build upon the following UI technologies.

  - Express for the node server
  - JSON to process the HTML
  - Vannila Javascript
 
# Prerequisites
  - npm
  - node

# Start Node server

 ```sh
$ cd FestivalSale
$ npm install
$ npm start
```
You can use npm start for MAC and npm start-win for Windows
